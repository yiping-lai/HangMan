# Hangman App

## Introduction
This is a website that users can play Hangman Game (https://en.wikipedia.org/wiki/Hangman_(game)). 

## About the Stack
Frontend: EJS, CSS, Javascript, jQuery
Backend: NodeJS

## Getting Started

## Installing Dependencies

#### NodeJS
Please install the latest version of Nodejs.

#### Npm packages

Install all npm packages by 

```bash
npm install 
```

## Running the server

To run the server, execute:

```bash
node appjs
```
